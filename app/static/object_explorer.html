3D Object Explorer

Purpose

This workspace stages real OBJ assets inside a bounded 3D world. The visible scene is the same spatial context the haptic pointer is expected to traverse in later hardware-backed builds.

Current Interaction

  • Load a bundled demo OBJ or a local OBJ file.
  • Apply a haptic material profile grounded in current desktop haptic capabilities.
  • Inspect the scene with orbit controls and move the pointer proxy with `W`, `A`, `S`, `D`, `Q`, and `E`.
  • Use `Space` or `Enter` near the object surface to emulate a contact event on the staged model.
