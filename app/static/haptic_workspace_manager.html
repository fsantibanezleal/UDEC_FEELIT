Haptic Workspace Manager

Purpose

This page manages structured desktop workspaces. A workspace file defines curated galleries and a safe file-browser root, while the heavy media stays in an external folder on the user's machine. The manager surfaces descriptor labels and registry state without exposing full local absolute paths by default.

Current Interaction

  • Create a new workspace rooted in an external folder.
  • Auto-populate that workspace from supported files already present in the chosen root.
  • Register an existing `.haptic_workspace.json` file.
  • Review the currently known workspaces before using them inside Haptic Desktop.
