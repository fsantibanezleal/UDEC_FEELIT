# FeelIT Asset Sources

## Purpose

This document records the origin of the bundled demo assets used in the current FeelIT object-explorer baseline.

## Bundled OBJ Models

### Walt Head

- local file: `app/static/assets/models/demo/WaltHead.obj`
- source: `three.js examples`
- upstream URL: `https://raw.githubusercontent.com/mrdoob/three.js/dev/examples/models/obj/walt/WaltHead.obj`
- use in FeelIT: contour-rich sculpture for facial relief exploration

## Asset Governance Rule

Bundled demo models are acceptable for internal prototyping, research, and technical validation. Before public redistribution or installer distribution, the final release process must verify the upstream licensing and attribution status of every bundled asset.
